### О приложении

Это серверная часть облака хранения данных с возможностью разграничения прав доступа к файлам.
В данном приложении использованы:

* #### Python-фреймворки:
    * Django 5.0.1
* #### Pre-commit хуки:
    * isort
    * pyupgrade
    * autopep8
    * flake8
* #### Базы данных:
    * MySQL
* #### Контейнеризация и оркестрация
    * Docker
### Запуск приложения
`docker compose up`
